The experimental folder contains stand-alone applications, demos, and features not yet ready for inclusion into the main app. We'd love to hear your feedback on these!

## Combined Workflows

* [Veo Variations](/vertex-ai-creative-studio/experiments/veo-variations) - Generates creative variations of a video concept and evaluates them technically using Nano Banana, Gemini, and Veo 3.1 Lite.
* [Countdown Workflow](/vertex-ai-creative-studio/experiments/countdown-workflow) - An automated two-stage pipeline to create branded countdown videos.
* [Creative GenMedia Workflow](/vertex-ai-creative-studio/experiments/creative-genmedia-workflow) - A workflow which takes various inputs to create prompts for Veo 3, plus a showcase of output videos.
* [Run, Veo, Run](/vertex-ai-creative-studio/experiments/run-veo-run) - A near-real time, multimodal video extension and generation experiment.

## Prompting Techniques

* [Promptlandia](/vertex-ai-creative-studio/experiments/promptlandia) - A powerful web application that helps you analyze, refine, and improve your prompts using AI-powered feedback.
* [Veo Genetic Prompt Optimizer](/vertex-ai-creative-studio/experiments/veo-genetic-prompt-optimizer) - An automated system that uses a genetic algorithm to evolve and refine high-level "metaprompts".
* [Veo 3 Character Consistency](/vertex-ai-creative-studio/experiments/veo3-character-consistency) - A workflow for generating a video of a person in a new scene while maintaining character consistency.
* [Veo 3 Item Consistency](/vertex-ai-creative-studio/experiments/veo3-item-consistency) - A workflow for generating a video of an item in a new scene while maintaining item consistency.

## Image Generation & Comparison

* [Imagen Product Recontextualization at Scale](/vertex-ai-creative-studio/experiments/imagen_product_recontext) - A set of notebooks for the Imagen Product Recontext model to run at scale.
* [Virtual Try-On](/vertex-ai-creative-studio/experiments/vto) - A notebook example for virtually trying on outfits at scale.
* [Brand Consistency](/vertex-ai-creative-studio/experiments/brand_consistency) - Maintain brand identity in generated media.
* [Arena](/vertex-ai-creative-studio/experiments/arena) - Rate your images within a visual arena.

## Audio & Voice

* [Babel](/vertex-ai-creative-studio/experiments/babel) - Experiment with Chirp 3 HD voices.

## Developer Tools & MCP

* [MCP GenMedia](/vertex-ai-creative-studio/experiments/mcp-genmedia) - Model Context Protocol servers for Genmedia services for AI applications to utilize Veo, Imagen, Lyria, Chirp 3 HD.
